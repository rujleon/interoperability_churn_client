    🚀 Guide de démarrage rapide

    Commencez à prédire le churn en 3 étapes simples.

    1. Enregistrez-vous pour obtenir votre token
    # POST /clients/register
curl -X POST https://your-api.com/clients/register \
  -H "Content-Type: application/json" \
  -d '{"name":"Akomba","email":"akomba@gmail.com","company":"Acme"}'
    2. Faites une prédiction
    # POST /predict
curl -X POST https://your-api.com/predict \
  -H "X-API-Token: churn_votre_token" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_name": "Akomba",
    "tenure_months": 8,
    "monthly_charges": 89.5,
    "total_charges": 716.0,
    "num_products": 2,
    "contract_type": 0,
    ...
  }'
    3. Interprétez la réponse
    {
  "churn_probability": 0.7842,
  "churn_prediction": true,
  "risk_level": "Élevé",
  "recommendation": "Action immédiate recommandée...",
  "key_factors": ["⚠️ Contrat mois-à-mois", "⚠️ Client récent"]
}
    🔐 Sécurité
    • ✅ Token HMAC-SHA256 par client
    • ✅ Expiration configurable du token
    • ✅ Rate limiting : 100 req/heure
    • ✅ Validation de signature à chaque requête
    • ✅ Renouvellement de token sécurisé
    • ✅ Isolation complète des données par client
